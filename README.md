# ERP SQL Toolkit

A practical SQL Server diagnostic tool built from real ERP support experience.

## About

This tool was created based on real-world ERP support scenarios.

During troubleshooting, support engineers and developers often need to quickly inspect databases, search tables, check structures, and gather information.

However, writing scripts repeatedly takes time.

ERP SQL Toolkit combines commonly used database inspection functions into one simple utility.

## Who is this for?

- ERP Support Engineers
- R&D Teams
- Database Administrators
- Developers

## Features

✅ SQL Server Database Overview  
✅ Table Search  
✅ Column Search  
✅ Schema Inspection  
✅ Database Information Checking  
✅ ERP Troubleshooting Support

## Why I built this

I created this tool to help technical teams save time during troubleshooting.

It is not designed to replace professional database analysis, but to provide a practical way to quickly gather information when time matters.
